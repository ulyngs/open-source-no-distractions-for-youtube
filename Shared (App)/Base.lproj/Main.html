This Safari extension hides distractions on YouTube. It is open source and collects zero data - you can find the source code on GitHub.

To turn it on, first open Safari and go to youtube.com.
Then, in the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on (Step 1).
