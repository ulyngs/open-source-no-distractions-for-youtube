This Safari extension hides distractions on YouTube (if you need it on more sites, go here). The extension is open source and collects zero data - you can see the source code on GitHub.

To turn it on, open Safari and go to youtube.com. Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow it to access youtube.com
