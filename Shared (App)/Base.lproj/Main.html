This Safari extension lets you hide distractions on YouTube (for any website, use MindShield).
The extension is open source and collects zero data - you can find the source code on GitHub.

To turn it on, open Safari and go to youtube.com. Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow it to access youtube.com
